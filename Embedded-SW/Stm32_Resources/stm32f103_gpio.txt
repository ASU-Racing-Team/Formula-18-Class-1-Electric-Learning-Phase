# General Overview
    ## Functionality
        Each pin can be
            Input floating 
            Input pull-up 
            Input-pull-down 
            Analog 
            Output open-drain 
            Output push-pull 
            Alternate function push-pull 
            Alternate function open-drain 
        And can run at 
            2 MHz max speed
            10 MHz max speed
            50 MHz max speed
    ## Registers
        Control Registers:
            GPIOx_CRL : Contol Registe Low
            GPIOx_CRH : Contol Registe High
       Data Registers:
            GPIOx_IDR : Input Data Register
            GPIOx_ODR : Ouput Data Register
        Set/reset Registers:
            GPIOx_BSRR : Bit Set Reset Register
            GPIOx_BRR : Bit Reset Register
        Locking Register:
            GPIOx_LCKR : Lock Register
